Google Calendar
Every new booking is pushed to this calendar automatically. To connect it:
  1. In Google Cloud Console, create a project, enable the Google Calendar API, and create a service account.
  2. Download the service account's JSON key and save it as google-service-account.json in the project folder (next to server.js).
  3. In Google Calendar, share your calendar with the service account's email with "Make changes to events".
  4. Paste the calendar's ID above (Calendar settings → "Integrate calendar"), save, then hit Test Connection.
The full walkthrough is in README.md.
